#### Les interfaces

- GND 
- 3V3 = pour l'alimentation des circuits
- GPIO 
- UART = avec les broches de type RX et TX
- ANALOGUE = avec les broches ADC2
- DAC 
- TOUCH 
- RTC 
- I2C = avec les broches SCL & SDA
- SPI = avec les broches de type VPSI & HSPI
